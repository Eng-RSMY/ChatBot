CHATBOT - LSTM - keras

-> Download the Google news word2vec , use this path : https://s3.amazonaws.com/dl4j-distribution/GoogleNews-vectors-negative300.bin.gz

A toy chatbot powered by deep learning and trained on dataset containing question and answer pair of different 19 categories:

* AI
* botprofile
* computers
* conversations
* emotion
* food
* gossip
* greetings
* health
* history
* humour
* literature
* money
* movies
* politics
* psychology
* science
* sports
* trivia

The dataset is contained in data folder each in yml file
